
# Log Analyze
A simple Python script to analyze log files and detect suspicious entries.

## Features
- Detects ERROR entries in log files
- Prints suspicious activity to the console

## How to Use
1. Clone the repository:
   `git clone https://github.com/YOUR_USERNAME/log-analyzer.git`
2. Navigate into the folder:
   `cd log-analyzer`
3. Run the script:
   `python analyze_logs.py`
4. Check the console output for suspicious entries
